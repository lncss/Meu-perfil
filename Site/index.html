Repositórios GitHub

Repositório Gitignore

Gitignore é uma coleção de modelos .gitignore úteis que funcionam bem com linguagens de programação, estruturas, ferramentas ou ambientes específicos.O repositório é muito bem organizado, sendo categorizado em modelos de pasta raiz, modelos globais para IDEs, ferramentas e sistemas operacionais e modelos de comunidade para casos de uso mais específicos.

Repositório APIs

ApIs públicas é um repositório extremamente útil que tem mais de 116.000 estrelas do GitHub. Ele faz a curadoria de uma lista de diferentes tipos de APIs que você pode usar como desenvolvedor web ou desenvolvedor de software em geral. Quase qualquer tipo de API que você precisará está disponível neste repositório.

Repositório JS

Este repositório é uma das populares e úteis séries de livros JavaScript de Kyle Simpson. Ele tem mais de 136.000 estrelas do GitHub e cobre as partes importantes do JavaScript que você precisa entender (escopo, fechamentos, classes, etc).
